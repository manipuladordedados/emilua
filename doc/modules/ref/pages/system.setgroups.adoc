ifeval::["{doctype}" == "manpage"]

== Synopsis

[source,lua]
----
local system = require "system"
system.setgroups(groups: table[number])
----

== Description

Sets the supplementary group IDs for the calling process.

NOTE: Only the master VM is allowed to use this function.
